# Figure Inventory

The figure files in `docs/figures/` are cleaned for manuscript use. In particular, the diagram images do not contain embedded `Figure 2`, `Figure 3`, or `Figure 5` captions. Captions should be generated by LaTeX or the target manuscript format.

Key files:

- `fig_scenario_decomposition_cropped.png`: scenario, dyads, and cell workflow.
- `fig_metric_evidence_chain_cropped.png`: metric rationale.
- `fig_framework_updated.png`: overall CCT-MARL framework.
- `fig_cct_data_flows_cropped.png`: decision, experience, and communication-safety data flows.
- `fig_shielded_baselines_response.png`: shield attribution summary.
- `fig_scalability_response.png`: scalability summary.
- `fig_reward_comm_response.png`: communication and reward sensitivity summary.
