# Metric definitions

- **Return**: cumulative episode reward. Higher is better.
- **Makespan**: number of time steps required to complete all construction cells. Lower is better.
- **Waiting**: number of waiting actions selected or induced during execution. Lower is better.
- **Collision**: number of detected spatial conflicts. Lower is better.
- **Rework**: number of rework operations. Lower is better.
- **Bits**: communication cost measured by transmitted message size. Lower is better under comparable task performance.
